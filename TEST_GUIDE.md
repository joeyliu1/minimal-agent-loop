# 功能验证手册

项目启动后，在交互模式下测试以下功能。

---

## 1. 数学计算 (MathTool)

**测试问题：**
```
You: 2+2等于几？
You: 15 * 25 + 100 是多少？
You: 计算 (100 + 200) / 3 的结果
```

**预期结果：** 返回正确的计算结果

---

## 2. 当前日期 (CurrentDateTool)

**测试问题：**
```
You: 今天几号？
You: 现在是几点？
You: 今天是星期几？
```

**预期结果：** 返回当前日期时间

---

## 3. 文件读取 (FileReadTool)

**前置准备：** 项目根目录存在测试文件

```bash
echo "This is a test file for reading." > /Users/lss/workspace/workspace-ai/minimal-agent-loop/test.txt
```

**测试问题：**
```
You: 读取 /Users/lss/workspace/workspace-ai/minimal-agent-loop/test.txt
You: 查看项目根目录下的 README.md 文件内容
```

**预期结果：** 返回文件内容

---

## 4. RAG 知识库 (RagTool)

### 4.1 添加单个文档

**测试问题：**
```
You: 把"Spring AI 是一个 AI 集成框架，它简化了 AI 应用的开发"加入知识库
You: 添加文档：人工智能（AI）是让计算机具有人类智能的技术，包括机器学习、深度学习等
```

**预期结果：** 提示文档添加成功

---

### 4.2 添加多个文档

**测试问题：**
```
You: 同时添加三条文档：1) Java 是一种面向对象的编程语言 2) Spring Boot 是 Java 微服务框架 3) Spring AI 是 AI 集成框架
```

**预期结果：** 提示添加了 3 条文档

---

### 4.3 查询知识库（带引用）

**测试问题：**
```
You: 什么是 Spring AI？
You: 查询知识库中关于人工智能的内容
You: 知识库里有什么关于 Java 的内容？
```

**预期结果：** 返回检索结果，并附有来源引用

---

### 4.4 验证引用功能

先添加文档：
```
You: 把"水的沸点是 100 摄氏度"加入知识库
```

再查询：
```
You: 水的沸点是多少度？
```

**预期结果：** 返回带有 `[source]` 引用的答案

---

## 常见问题

| 问题 | 解决方法 |
|------|----------|
| 工具调用失败 | 检查环境变量是否正确设置 |
| RAG 返回空结果 | 确认已添加过文档，知识库非空 |
| 文件读取失败 | 检查文件路径是否存在，路径需为绝对路径 |
